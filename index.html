My key Projects

By the time , I have done some of the projects based on Machine Learning , Artificial Intelligence, Database Management System , Web Development , APIs etc. All the projects are uploaded on my github profile . YOu can view the source code by clicking the button below .

Sign Language Detector

A model that can recognize and interpret various sign language gestures one based on LSTM and another using Teachable Machine. The LSTM model allows real-time hand sign detection through your computer's camera.Teachable Machine utilizes a pre-trained model generated using your custom dataset of hand sign images. Both models are capable of real-time sign language detection.

Lensify Application

Lensify is a Streamlit web application that identifies landmarks from uploaded images using a TensorFlow model. It provides detailed information about the predicted landmark, including location, significance, and history, and displays the location on a map. Users can also navigate to the landmark using a provided Google Maps direction link.

Chatbot

This model is built using FastAPI and Dialogflow. It handles various functionalities such as adding items to an order, removing items from an order, completing an order, and tracking an order status. The chatbot interacts with a MySQL database to store and retrieve order information.

Emotion Predictor

This project uses OpenCV and Keras to detect human emotions in real-time through a webcam. The model predicts emotions such as Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise.The face is detected using the Haarcascade classifier. It uses Python 3.x, OpenCV, Keras, TensorFlow, NumPy
